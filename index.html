💰 펀딩 / 상금 구조

  • 아빠 50,000원, 엄마 50,000원
  • 다연 30,000원, 성연 30,000원
  • 총 펀딩 160,000원
  • 1위 100,000원, 2위 60,000원
  • 3위와 4위는 상금 없음

🔒 예측 공개 룰

  • 경기 시작 전까지 승/패 예측 가능
  • 경기 시작 전에는 다른 가족의 선택 비공개
  • 경기 시작 전에는 입력 여부만 표시
  • 경기 시작 후 전체 예측 공개
  • 경기 시작 후 예측 수정 불가

🏆 점수 룰

  • 승패 적중: +1점
  • 3연속 적중: +1점 추가
  • 5연속 적중: +2점 추가
  • 3연속 실패: +0.5점 추가
  • 5연속 실패: +1점 추가
  • 6연속 이상은 추가 보너스 없음
  • 취소 경기 / 결과 미입력 경기는 점수 계산 제외

⚙️ 운영 룰

  • 월초에 관리자가 한 달 경기 일정을 등록
  • 경기 종료 후 관리자가 결과 입력
  • 월 단위로 순위와 상금 정산
  • 동점 시 기본 정렬은 아빠 → 엄마 → 다연 → 성연
